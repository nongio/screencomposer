# Contributing

Screencomposer welcomes contributions from everyone. If you're interested in getting involved, here are some tips to help you get started.

## Coordination

Please use GitHub for discussions about features and their implementations. If you have questions, suggestions, or ideas, you can either open a new issue or contribute to an existing one if it’s relevant to the topic.

## Scope

The project README contains a rough roadmap that you can follow, or you can explore the compositor and suggest changes that you find useful. Before starting any contributions, please open an issue to discuss your ideas.
